### Commands:

```sh

cd app
npm init -y


npm install -g typescript # for global installation and we can use the `tsc -b` directly
tsc --init
tsc -b
# vs
npm install typescript --save-dev # for local installation and we can use the `npx tsc -b` as its local 
npx tsc --init
npx tsc -b


npm intall -g ts-node
ts-node index.ts
# or
npm intall ts-node --save-dev
npx ts-node index.ts

```
